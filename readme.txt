ｅ支店APIのサンプルコード（python）

ファイル名: e_api_get_master.py

APIバージョン： v4r3で動作確認

機能：ログイン、マスターダウンロード、ログアウト。
（おまけ：read_master.py、営業日の読み出し。）

ご注意！！ ================================

	APIで本番環境に接続した場合、実際に注文を出せます。
	市場で条件が一致して約定した場合、取り消せません。
	十分にご注意いただき、ご利用ください。

=========================================


1）動作テストを実行した環境は、os: Centos7.4、python: 3.6.8 です。

２）事前に立花証券ｅ支店に口座開設が必要です。

３）利用時に変数を設定してください。

	コードの開始部分の「利用時に変数を設定してください」のセクションで、
	変数、url_base、my_userid等を実際のurl、
	ご自身のユーザーID、パスワード等に変更してください。
	変更しない場合、APIを利用できません。


４)実行はコマンドプロンプト等からpython環境で起動してください。マスターのダウンロードはシステム負荷の高い動作なので、立会時間外での実行をお願い致します。


５）実行内容は、以下になります。

	5-1）ログインして仮想URL（request）と仮想URL（event）を取得します。

	5-2）マスターをダウンロードし、指定のファイル名で保存する。

	5-3) ログアウトする。
	
	5-おまけ) おまけの read_master.py で営業日の読み出しが出来ます。

６）利用時間外に接続した場合、"p_errno":"9"（システム、サービス停止中。）が返されます。

	詳しくは「立花証券・ｅ支店・ＡＰＩ、EVENT I/F 利用方法、データ仕様」4ページをご参照ください。
	なおデモ環境は、メンテナンス待ちで現在ご利用いただけません。
